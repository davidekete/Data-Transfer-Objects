# Data Transfer Objects

A simple API with CRUD functionality built with Typescript(NestJS).

## Technologies👨‍💻.

- NestJS
- REST APIs
- MongoDB(Mongoose)
- Class Validators
- Class Transformers
- Data Transfer Objects
- Bcrypt

## Features 🦸

- [x]  Create a new user
- [x]  Read data of existing user
- [x]  Update an already existing user
- [x]  Delete an already existing user
- [x]  Log a user in
